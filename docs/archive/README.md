# 歷史文件封存區

此目錄保存過去版本、已撤銷方向與產品演化資料。

## 文件效力

本目錄內的文件：

- 只供追溯歷史決策與理解產品演化
- 不得作為目前產品需求、開發命令或驗收依據
- 不得因文件中出現「正式」、「凍結」、「不得修改」等舊文字，就凌駕目前控制文件
- 與現行規格衝突時，一律以 `docs/control/` 內的正式文件為準

## PMS 文件

PMS 與目前的生活管理 App 已正式分開。封存在此處的 PMS 文件保留歷史價值，但沒有現行產品控制力，也不得把生活管理 App 再帶回責任承接系統、抽象入口或 PMS 規格主線。
